# Golfside Luxury Hotel Web Platform

A premium, scalable, cloud-first hotel management and booking platform built with Next.js App Router, Prisma ORM, and TailwindCSS.

## Overview
This platform acts as both the **public facing luxury brochure** for the hotel and the **internal operational management dashboard**. 

**Key Features:**
- Database-driven content editing (NO hardcoded text / phones / emails).
- Protected administrative dashboard for check-ins, tracking, and metric monitoring.
- Robust schema handling Guests, Bookings, Room Categories, Amenities, and Site Rules.
- Ready for Vercel Deployment.
- Luxury Gold / Dark Mode native UI.

Refer to `DEPLOYMENT.md` for infrastructure handling on the cloud.
